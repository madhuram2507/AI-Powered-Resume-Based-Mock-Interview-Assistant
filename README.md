# AI-Powered Interview Evaluation System

An end-to-end AI-based interview platform that evaluates candidate responses using voice input and Large Language Models.

## 🚀 Features
- Voice-based interview answers using browser MediaRecorder API
- Speech-to-text processing for candidate responses
- AI-driven answer evaluation and automated scoring
- Detailed feedback generation
- FastAPI backend with REST APIs
- Frontend built using HTML, CSS, and JavaScript

## 🏗️ Tech Stack
- Python
- FastAPI
- JavaScript (Vanilla)
- HTML5, CSS3
- Speech-to-Text
- Large Language Models (LLMs)
